##  Analisis de datos con Python

#### Descripción
<p>Este repositorio contiene código para realizar web scraping en páginas de tiendas, extrayendo información sobre productos y almacenándola en un archivo CSV.</p>
<p>Además, se lleva a cabo un análisis de datos sobre estos productos utilizando las bibliotecas Python Pandas, NumPy y Natsort.</p>

## Acerca de las librerias implementadas

##### 1. Pandas
###### Descripción:
<p>Una potente biblioteca para manipulación y análisis de datos, proporcionando estructuras de datos flexibles y herramientas para trabajar con conjuntos de datos tabulares.</p>

##### 2. NumPy
###### Descripción:
<p>Utilizado para realizar operaciones numéricas eficientes en matrices y arreglos, fundamental para el análisis numérico.</p>

##### 3. Natsort
###### Descripción:
<p>Facilita el ordenamiento "natural" de cadenas de texto, siendo útil en la ordenación de datos, como en la clasificación de nombres de productos.</p>
